## b. Konsep Grafika yang Digunakan
Program ini merupakan simulasi visualisasi molekul air (H₂O) yang menerapkan konsep dasar **grafika komputer 3D**. Molekul air divisualisasikan menggunakan bentuk geometri sederhana (lingkaran/sphere) untuk merepresentasikan atom Oksigen dan Hidrogen, dengan susunan **bengkok (bent)** sesuai struktur molekul aslinya.

Konsep grafika komputer yang digunakan dalam program ini meliputi:
- **Transformasi Geometri**, meliputi translasi, rotasi, skala, dan refleksi objek.
- **Proyeksi Perspektif**, untuk menampilkan objek 3D ke dalam tampilan 2D.
- **Rotasi Otomatis (Auto-Rotate)**, untuk memperlihatkan molekul dari berbagai sudut pandang.
- **Interaksi Keyboard**, sebagai kontrol pengguna untuk:
  - Rotasi objek
  - Zoom in dan zoom out
  - Reset tampilan
  - Mengganti jenis molekul (melalui tombol 1–5)

Selain itu, program juga menampilkan **informasi molekul** seperti struktur, sudut ikatan, massa molekul, jumlah atom, dan ikatan untuk membantu pemahaman konsep kimia secara visual.

## c. Cara Menjalankan Program
1. Pastikan compiler atau environment yang dibutuhkan sudah terpasang (misalnya OpenGL/GLUT atau library grafika yang digunakan).
2. Buka folder proyek.
3. Jalankan file program utama sesuai dengan bahasa pemrograman yang digunakan.
4. Setelah program berjalan, gunakan keyboard untuk berinteraksi:
   - Tombol panah: rotasi objek
   - Tombol `+` / `-`: zoom in dan zoom out
   - Tombol `R`: reset tampilan
   - Tombol `P`: mengubah mode proyeksi
   - Tombol `G`: menampilkan grid
   - Tombol `A` / `Space`: mengaktifkan rotasi otomatis
   - Tombol `1–5`: mengganti molekul

---

Program ini dibuat sebagai media pembelajaran untuk memahami **konsep grafika komputer** sekaligus **visualisasi struktur molekul dalam kimia**.
